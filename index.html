Proyecto de análisis y comparación salarial

Esta aplicación web permite explorar, comparar y comprender distintos indicadores relacionados con salarios, formación, sectores económicos, países y mercado laboral.

Ir a comparadores Ver conclusiones

📊 Objetivo del proyecto

El objetivo principal es transformar datos estadísticos en una herramienta visual, interactiva y fácil de utilizar, permitiendo realizar comparaciones entre diferentes variables socioeconómicas.

🧠 Análisis realizado

A partir de distintas bases de datos se ha realizado un análisis exploratorio para extraer patrones, diferencias y conclusiones relevantes sobre salarios, formación, empleo y desigualdades.

🖥️ Aplicación web

La web incluye comparadores interactivos y páginas de conclusiones que permiten consultar los resultados del análisis de forma clara.
